# Studien-Dashboard

Dieses Projekt ist ein lokales Dashboard zur Darstellung des Studienfortschritts.

## Start

1. Abhängigkeiten installieren:

```bash
python -m pip install -r requirements.txt
```
2. Anwendung starten:
```bash
python app.py
```

3. Dashboard im Browser öffnen:

http://127.0.0.1:8050
